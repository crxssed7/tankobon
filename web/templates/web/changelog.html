Site Changelog

31st August 2022

You may have noticed that the URL for this site has changed. This was due to an issue with the host provider and has required some minor changes in the backend. These widgets are subject to design changes.

  • Added widget for manga. Go to /widget to see the widget for specific manga.
  • Minor backend tweaks

19th August 2022

  • Query parameter for manga details page called volume that will open a specific volume by default. Example: http://127.0.0.1:8000/manga/4/?volume=1
  • Manga page now has a sticky details section so it is always shown on screen whilst scrolling

11th August 2022

The main update for this log is the ability to import data from third party sites.

  • Third party site imports
    • Kitsu
    • AniList
  • Changed the date field for any new manga to a date picker instead of free text
  • Addition of site changelog
