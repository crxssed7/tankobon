API Documentation

Tankōbon has a simple GET API that you can use to integrate with your own applications. All requests sent to the API should be sent with the GET method.

Get a list of Manga
Endpoint: /api/manga/

Query Parameters:

Name Default Description
offset 0 Offsets the results, for pagination.
limit 8 Amount of results to return
q Query string to filter by. Filters against the manga name and romaji.
sort name Sorts the results. The allowed values are last_updated and name
status Filter by status. The allowed values are releasing, finished and releasing
